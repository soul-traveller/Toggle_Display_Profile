# Toggle\_Display\_Profile — User Guide
**Version:** 1.2<br>
**Platform:** macOS<br>
**Author:** Knut Larsson<br>
<br>
Quickly switch between display ICC profiles from a list using ArgyllCMS (dispwin).

`Toggle_Display_Profile` is a simple Bash command script that simplifies selecting and applying a set of ICC profiles on macOS.<br>
